# news_viewer


## axios로 API 호출해서 데이터 받아오기 
```javascript
yarn add axios
```

<br>

## prettierrc 생성 
.prettierrc
```javascript
{
    "singleQuote" : true,
    "semi" : true,
    "useTabs" : false,
    "tabWidth" : 2,
    "trailingComma" : "all",
    "printWidth" : 80
}
```

<br>

## 파일 자동 불러오기 기능 활성화 
jsconfig.json
```javascript
{
    "compilerOptions" : {
        "target" : "es6"
    }
}
```
